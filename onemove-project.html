Project Overview

For NDP’25, I collaborated closely with my peers to develop onemove, a web and mobile event management platform designed to support large-scale event operations. The platform provides features such as real-time location tracking, logistics monitoring, schedule management, and coordination tools for different user groups. I contributed ideas and was responsible for designing and developing the UI/UX, ensuring the application was intuitive, accessible, and user-friendly for people of all ages.

Key Features

  • Live Location Tracking: Real-time tracking of personnel, vehicles, and equipment
  • Event Status Dashboard: Monitor ongoing events, crowd density, and road conditions
  • Logistics Management: Track props, equipment, and resources
  • Ticket Management: Verify and track event tickets
  • CET/TTX Integration: Manage Command Exercise Training (CET) and Table-Top Exercise (TTX) schedules
  • Role-Based Access Control: Different access levels for users, admins, and super admins
